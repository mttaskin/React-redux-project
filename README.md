# Redux Toolkit example

## `setup`

```
npm install veya yarn
```

## `using library`

- `@reduxjs/toolkit`
- `react-redux`
- `axios`
- `react-router-dom`
- `@mui/material-ui`
- `@emotion/react`
- `@emotion/styled`
